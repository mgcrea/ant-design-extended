  <CheckboxGroup defaultValue={["apple"]} {...props}>

    <CheckboxButton value="orange">Orange</CheckboxButton>

# Checkbox

## Button

A new `Checkbox.Button` component is available to bring button styles to checkboxes

<Canvas>
  <Story name="Button" argTypes={{ size: { control: { type: null } } }}>
    {(args) => (
      <Block>
        <Checkbox.Button value="apple">Apple</Checkbox.Button>
      </Block>
    )}
  </Story>
</Canvas>

## Sizes

`<Checkbox.Group />` does support a new `size` prop that matches the `SizeProps` of the extended `<Button />` component.

<Canvas>
  <Story name="Sizes" argTypes={{ size: { control: { type: null } } }}>
    {(args) => (
      <Block direction="column">
        <small>x-small</small>
        {CheckboxButtonGroupTemplate({ ...args, size: "x-small" })}
        <small style={{ marginTop: "1em" }}>small</small>
        {CheckboxButtonGroupTemplate({ ...args, size: "small" })}
        <small style={{ marginTop: "1em" }}>medium</small>
        {CheckboxButtonGroupTemplate({ ...args, size: "medium" })}
        <small style={{ marginTop: "1em" }}>large</small>
        {CheckboxButtonGroupTemplate({ ...args, size: "large" })}
        <small style={{ marginTop: "1em" }}>x-large</small>
        {CheckboxButtonGroupTemplate({ ...args, size: "x-large" })}
      </Block>
    )}
  </Story>
</Canvas>
